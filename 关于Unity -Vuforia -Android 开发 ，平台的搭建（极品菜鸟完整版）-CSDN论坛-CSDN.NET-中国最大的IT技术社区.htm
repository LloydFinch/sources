一、首先安装 java jdk , 度娘 “JDK” 进入官网下载即可,链接如下：

http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html  

同意协议之后方可下载，版本建议使用最新版。
下载完之后，是这样的，我下的是java 7的版本，其实其他版本也可以的。

接下来需要安装它，建议装到C盘，然后为其配置环境变量。

环境变量如下：

新建  系统变量   JAVA_HOME 和 CLASSPATH ,还有Path.
变量名：JAVA_HOME 
变量值：C:\Program Files\Java\jdk1.7.0
变量名：CLASSPATH 
变量值：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;
变量名：Path 
变量值：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
完成这些之后，
cmd -> java
cmd -> javac
cmd -> java -version

如果显示对应的文字信息，就说明你已经配置成功了。

二、到这来是第二步，安装google-android-sdk ,度娘 android SDK 英文不好的同学请看下图:

请确定你的电脑是32位还是64位的，按版本下载。。。。（这个是包括ADT 开发工具的，内含eclipse,java开发工具，如果你不需要请看下面）

这个就是 单纯 的android -SDK了，红色箭头指向的是 官网建议您最好使用安装版。
下载完成之后，打开它，安装即可，建议安装到C盘，必要的话，减少目录层，不然套很多，配置环境变量就恶心了。。

环境变量配置(同样是下面的系统变量): 
变量名:ANDROID_SDK_HOME
变量值:android-SDK-windows的路径    （路径，请在自己的C盘目录下面找清楚）

然后原有的Path 后面添加 
说明：
（%ANDROID_SDK_HOME% = 你的SDK 目录的引用 ）  不明所以的同学，主要这一点。
 后面是目录的位置。。。
%ANDROID_SDK_HOME%\platform-tools;%ANDROID_SDK_HOME%\tools; 

完成这些之后，可以测试一下：
 cmd -> adb 
 cmd -> android 

“adb” 会出现ADB的详细信息。
“android” 会出现一个更新SDK的窗口。 

**在这里请注意了，因为谷歌的安卓SDK更新时用的链接比较特殊，会导致有些更新失败，解决方法已经找到了，如下**

在你电脑里找到这个目录的这个文件，右键，按记事本的方式它，在最后另起一行加上如下一句话：

74.125.237.1 dl-ssl.google.com  

然后 保存，保证你更新速度咔咔的。  尽量多选一些API版本。 其他的保持默认。

三、第三步了，安装 Unity 及导入高通 AR SDK 开发包 

这里相对简单一些，去Unity 官网，Unity3d.com   点DOWNLOAD 你会看到下面这样：

箭头指向，是历史版本。

你是Windows请点win, 我用的是Unity4.1.5版本，新版本破解方法出的会慢一些。

下载下来安装，没神马好说的。
安装完成之后运行，现在30天试用版本，注册一个Unity账号登陆即可。

之后去高通官网下载 AR -sdk-Unity 
https://developer.vuforia.com/resources/sdk/android
下载的时候注意选择Unity, 它是一个.unitypackage 文件 ，这个是打开Unity 然后导入的。

当下载完 高通 支持 Unity 的　ＳＤＫ之后，它是个．ｕｎｉｔｙｐａｃｋａｇ的一个文件。
